# sshkey

rgb 颜色与 hex 颜色相互转换，为 IOS 的RGB colorWithRed 方法提供方便。

---

## 1. 安装

### 从 npm 安装

    $ npm install rgbhex -g

### 从 Github 安装

    $ git clone git@github.com:silentcloud/rgbhex.git
    $ cd rgbhex
    $ npm install -g

## 3. 实例

    $ rgbhex 255,255,255

    $ rgbhex f11

    $ rgbhex #f1f1f1

    $ rgbhex #f11
